# Homework_Business_analytics
Questa repository contiene un abbozzo di libreria per la simulazione di processi modellabili tramite reti di Jackson che permette l'analisi del comportamento del sistema e il calcolo di alcune statistiche rilevanti.  
L'homework è stato sviluppato all'interno dell'esame _Business Analytics_ (Politecnico di Torino, 2024/2025).
